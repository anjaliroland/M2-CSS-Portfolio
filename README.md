# M2-Portfolio

## Description

For this Module we were challenged to create a site that acts as a portfolio of work samples that potential employers can look at to easier asses if we are well suited for a potentiial job. Using what we learned the last few weeks in class I was able to use CSS and HTML to create a site that meets the following acceptance criteria:
<ul>
<li>When my portfolio is loaded the user can see my name, a recent photo, and links to an about me seection, my work, and a contact section.
<li>The UI scrolls to the corresponding section when the user clicks a link in the nav bar.
<li>When clicking on the work section the UI scrolls to a section with titled images of my projects (they are fake projects to act as placeholders until I have projects I can display).
<li>The first project shown is larger in size to the other projects in the work section.
